# 介绍

**Clipboard Manager** 是一个基于 **Rust (Tauri v2)** 和 **Vue 3** 构建的轻量、安全且现代化的剪贴板管理工具。

它旨在帮助用户高效地管理复制历史，支持文本和图片，并提供强大的搜索和隐私保护功能。

## ✨ 核心特性

### 📋 全面的历史记录

自动记录您复制的所有内容，无论是代码片段、重要文本还是截图。再也不用担心丢失刚才复制的信息。

### 🔍 极速智能搜索

内置高性能全文搜索引擎。只需输入关键词，即可毫秒级定位到您需要的历史记录。

### 🖼️ 图片预览与管理

不仅支持文本，完美支持图片格式。您可以在列表中直接预览复制的图片，并一键再次复制使用。

### 🔒 隐私安全优先

我们深知剪贴板数据的敏感性：

- **本地存储**: 所有数据仅存储在您的设备本地 (`~/.clipboard-manager/`)，绝不上传云端。
- **加密存储**: 数据库采用 AES-GCM 高级加密标准，确保数据即使被盗也无法读取。
- **敏感过滤**: 自动识别密码等敏感信息，并支持设置忽略特定应用（如 1Password）的复制行为。

### ⚡ 极致性能体验

- **Rust 驱动**: 后端采用 Rust 编写，内存占用极低，响应速度极快。
- **懒加载优化**: 针对大量历史记录和大文本进行特殊优化，确保界面始终流畅。

### ⌨️ 键盘党福音

- **Vim 风格导航**: 支持 `j` / `k` 上下移动，`Ctrl+n` / `Ctrl+p` 快速选择。
- **全局快捷键**: 无论在哪个应用中，只需按下快捷键（默认 `Cmd+Shift+V`）即可唤出面板。

### 🚀 高级生产力

- **粘贴队列 (Paste Stack)**: 复制多个项目并按特定顺序依次粘贴。非常适合填写表单。
- **OCR 文字识别 (macOS)**: 直接从剪贴板历史记录中的图片提取文字。

## 🌍 跨平台与国际化

- 支持 macOS, Windows 和 Linux。
- 内置简体中文和英文界面，自动跟随系统语言。
