# 使用指南

## 基础操作

### 唤出界面

默认情况下，您可以使用以下全局快捷键唤出剪贴板历史面板：

- **macOS**: `Cmd + Shift + V`
- **Windows / Linux**: `Ctrl + Shift + V`

> **提示**: 您可以在应用的设置页面修改此快捷键。

### 列表导航

为了提高效率，我们支持多种导航方式，让您的双手无需离开键盘：

- **上下移动**:
  - 方向键 `↑` / `↓`
  - Vim 风格 `j` (下) / `k` (上)
  - Emacs 风格 `Ctrl+n` (下) / `Ctrl+p` (上)
- **确认/粘贴**: 按下 `Enter` 键将选中的内容粘贴到当前活动窗口，并自动关闭面板。
- **预览详情**: 按下 `Space` (空格键) 查看当前选中项的详细信息（适用于长文本或大图）。
- **关闭面板**: 按下 `Esc` 键。

## 搜索功能

面板顶部设有搜索框。打开面板后，直接输入关键词即可过滤历史记录。

- 支持模糊搜索。
- 搜索范围包括文本内容和来源应用名称。

## 数据管理

### 存储位置

为了保障您的数据隐私，所有数据均存储在您的本地设备上：

- **macOS**: `~/.clipboard-manager/`
- **Windows**: `C:\Users\<Username>\.clipboard-manager\`
- **Linux**: `~/.clipboard-manager/`

### 数据库安全

- 数据库文件名为 `history.db`。
- 数据库使用 **AES-GCM** 算法进行加密，密钥存储在本地安全位置。这意味着即使有人获取了您的数据库文件，没有密钥也无法读取其中的内容。

### 隐私设置

在设置中，您可以：

- **暂停记录**: 暂时停止监听剪贴板变化。
- **忽略应用**: 添加不需要记录的应用名称（如 `1Password`, `KeyChain` 等），保护敏感信息。
- **清空历史**: 一键删除所有本地存储的历史记录。
